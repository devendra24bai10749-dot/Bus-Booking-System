#  Bus Ticket Booking System (Java)

##  Overview

The **Bus Ticket Booking System** is a Java-based console application designed to automate the process of bus seat reservation. It eliminates manual booking issues like seat duplication, confusion, and lack of tracking.

This project demonstrates core **Object-Oriented Programming (OOP)** concepts along with real-world system design.

---

##  Problem Statement

Traditional bus booking systems often suffer from:

* Manual seat allocation errors
* Lack of real-time seat availability
* No proper booking record management
* Time-consuming ticket handling

---

##  Solution

This system provides a digital solution where users can:

* View available buses and seats
* Book tickets instantly
* Cancel bookings safely
* Track all reservations

---

##  Features

###  Bus Management

* Predefined bus routes
* Source → Destination display
* Fare information

### Booking System

* Seat selection
* Passenger name registration
* Unique booking ID generation

###  Cancellation System

* Cancel booking using Booking ID
* Automatic seat release

###  Records Management

* View all active bookings
* Track seat availability
